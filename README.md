# TOONHUB Landing

A full-viewport character-figurine carousel hero section built with React 19 + TypeScript + Vite 8 + Tailwind CSS 4.

## Features

- 4-figure carousel with role-based positioning (center, left, right, back)
- Smooth 650ms cubic-bezier crossfans for background color, scale, blur, opacity
- SVG grain noise overlay
- Giant "3D SHAPE" ghost text with Anton font
- Fully responsive (mobile-first, 360px+)
- Preloaded images for zero-latency swaps

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4 (`@tailwindcss/vite`)
- Lucide React (ArrowLeft, ArrowRight icons)
- Google Fonts: Anton + Inter

## Setup

```bash
npm install
npm run dev
```

## Build

npm run build
